Gemini's Personal Intelligence beta, launching in the US, connects Google apps to personalize user experiences by securely linking Gmail, Photos, YouTube, and Search for tailored assistance. [### Understanding Manus sandbox - your cloud computer (7 minute read)](https://manus.im/blog/manus-sandbox?utm_source=tldrai)

Manus Sandbox is a fully isolated cloud virtual machine that runs in its own environment. Manus allocates a sandbox for each task and can execute sandboxes in parallel. Sandboxes have full capabilities, including networking, file systems, a browser, and various software tools. They can work around the clock to complete assigned tasks without consuming local resources.

### Engineering & Research

[### How Shopify and Google built the Universal Commerce Protocol (UCP) so AI agents can transact with any merchant (Sponsor)](https://shopify.engineering/UCP?utm_source=tldrai)

Shopify and Google co-developed an open standard for AI agents: the [Universal Commerce Protocol (UCP)](https://shopify.engineering/UCP). UCP defines the discovery and negotiation mechanisms between agent and merchant, and makes it simple for humans and agents to work together. [Read more on the Shopify Engineering Blog](https://shopify.engineering/UCP)

[### Tool Search now in Claude Code (2 minute read)](https://x.com/trq212/status/2011523109871108570?utm_source=tldrai)

MCP Tool Search is now rolling out in Claude Code. The feature allows Claude Code to dynamically load tools into context. Claude Code detects when MCP tool descriptions would use more than 10% of the context, and when triggered, tools are loaded via search instead of preloaded. MCP tools will otherwise work exactly as before.

[### GPT-5.2-Codex is now available in the Responses API (2 minute read)](https://threadreaderapp.com/thread/2011499597169115219.html?utm_source=tldrai)

GPT-5.2-Codex is an upgraded version of GPT-5.2 optimized for agentic coding tasks in Codex and similar environments. It supports four levels of reasoning effort settings and has a 400,000-token context window. The model costs $1.75 per million input tokens and $14.00 per million output tokens. It is now available in the Responses API.

[### Ministral 3 Technical Report (18 minute read)](https://arxiv.org/pdf/2601.08584?utm_source=tldrai)

Ministral 3 is a new family of dense language models (3B, 8B, and 14B) optimized for low-resource environments, offering base, instruction-tuned, and reasoning variants. The models support image understanding and were trained using Cascade Distillation, an iterative distillation and pruning method.

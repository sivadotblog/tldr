ChatGPT's market share has dipped below 50% for the first time. Users are migrating to other assistants like Google's Gemini, Anthropic's Claude, and xAI's Grok. ### Deep Dives & Analysis

[### Kimi K2.7 Code vs Claude Fable 5: Landing Pages That Cost 94% Less (6 minute read)](https://x.com/nutlope/status/2067281915887943890?utm_source=tldrai)

An experiment compared Kimi K2.7 Code with Claude Fable 5 to generate 12 landing pages each. Kimi K2.7 Code cut costs by 94%, being 16 times cheaper than Claude Fable 5.

[### Brain the Size of a Planet: Are LLMs Thonking too Hard? (30 minute read)](https://parsiya.net/blog/llm-thonking/?utm_source=tldrai)

Earlier studies showed that Opus 4.6, GPT 5.4, Gemini 3.1-pro-preview, Deepseek R1-0528, and Qwen 3.6-plus were unable to find two of the vulnerabilities discussed in the Mythos blog post without extremely revealing hints. This study continues the previous experiments with 26 distinct Claude-4.6/4.7 and GPT-5.4/5.5 combinations and different context window sizes and reasoning efforts. It found that higher reasoning effort, and even later models, are not always better for triaging security results.

[### Production Infrastructure for AI Agents (19 minute read)](https://vercel.com/blog/introducing-eve?utm_source=tldrai)

Vercel introduced eve, an open-source agent framework with built-in capabilities such as durable execution, sandboxed compute, approvals, subagents, and evaluations. It aims to let developers focus on agent behavior while the framework handles production concerns.

[### MolmoMotion: Language-guided 3D motion forecasting (9 minute read)](https://huggingface.co/blog/allenai/molmomotion?utm_source=tldrai)

MolmoMotion, a new motion forecasting model, predicts future 3D point trajectories in videos using language instructions and initial object positions, outperforming existing methods. The model and its dataset, MolmoMotion-1M, feature vast datasets of 3D point trajectories from 1.16M videos, with a benchmark for accuracy testing.

[### Vercel Connect (8 minute read)](https://vercel.com/blog/introducing-vercel-connect?utm_source=tldrai)

Vercel launched Connect in public beta, replacing long-lived provider tokens with runtime credential exchange that issues short-lived, task-scoped credentials for agents. The approach reduces the risk associated with shared, persistent access tokens.

[### Building AI Agents for AR Glasses and XR Devices with NVIDIA XR AI (12 minute read)](https://developer.nvidia.com/blog/building-ai-agents-for-ar-glasses-and-xr-devices-with-nvidia-xr-ai/?utm_source=tldrai)

Nvidia XR AI provides a reusable foundation for connecting extended reality devices to GPU-accelerated AI services running in the cloud, data center, workstation, or edge. It gives developers access to an open source library for building intelligent agents for AI glasses, AR glasses, and XR headsets. These agents will be able to see what users see, understand token or typed intent, call enterprise tools, and respond within the same XR session. XR AI is now available in public beta.

[### Loop Engineering (13 minute read)](https://x.com/addyosmani/status/2064127981161959567?utm_source=tldrnewsletter)

Loop engineering is when a developer replaces themselves as the person who prompts the agent with a system that does it instead. The loop is a recursive goal where the user defines a purpose and the AI iterates until complete. The technology is still early - users have to be careful about token costs and still monitor to ensure quality - but this is likely the future of how people work with coding agents. Stop prompting coding agents and design loops that prompt agents instead.

[### Core AI (Website)](https://developer.apple.com/documentation/coreai/?utm_source=tldrnewsletter)

Apple's Core AI framework allows developers to build, run, and deploy AI models in apps on Apple silicon. It enables access to the latest model architectures and inference techniques across the CPU, GPU, and Neural Engine. The Swift API gives developers control over model specialization, caching, and inference performance while making common tasks simple. Core AI integrates with Xcode and the developer toolchain.

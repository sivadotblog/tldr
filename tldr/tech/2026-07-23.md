[### Science Corporation's vision-restoring chip wins EU approval (5 minute read)](https://techcrunch.com/2026/07/22/science-corporations-vision-restoring-chip-wins-eu-approval/?utm_source=tldrnewsletter)

Science Corporation has won approval from Europe's medical device regulator to begin selling its PRIMA device, which restores vision lost from age-related macular degeneration. The device has also received a designation from the US FDA, the first step toward an expedited regulatory review. It requires an hour-long outpatient procedure that plants a small chip in the back of the eye. [### Durable Objects are Made for Agents (11 minute read)](https://calv.info/durable-objects-are-made-for-agents?utm_source=tldrnewsletter)

Cloudflare Durable Objects are a primitive that combines a serverless V8 isolate for running code, a paired SQLite instance, and request routing to a particular isolate. They allow developers to start thinking about their service as a set of many event-driven objects rather than as a bunch of services talking to a database. Durable Objects run explicitly in a single-threaded manner and come with native support for WebSockets. They are cheap and extremely token-efficient.

[### The State of WebMCP (16 minute read)](https://www.spronta.com/blog/state-of-webmcp-july-2026/?utm_source=tldrnewsletter)

WebMCP lets web pages register typed JavaScript functions that browser AI agents can call. While there is strong industry support behind the spec, site adoption is close to zero. No mainstream agent consumes the tools yet, but Google says Gemini in Chrome will be the first. The API changed locations in a recent draft, which means early adopters already carry migration debt.

Britain's DragonFire laser weapon shot down drones flying at 404 mph on November 20 at the Ministry of Defence's Hebrides Range in Scotland. The high-powered, solid-state laser will be installed in the Navy's Type 45 frigates by 2027. It can hit a target the size of a US quarter over a distance of 0.62 miles. There is a security vulnerability in React Server Components that allows unauthenticated remote code execution. It exploits a flaw in how React decodes payloads sent to React Server Function endpoints. Apps may still be vulnerable even if they don't implement any React Server Function endpoints. The vulnerability is present in versions 19.0, 19.1.0, 19.1.1, and 19.2.0 of react-server-dom-webpack, react-server-dom-parcel, and react-server-dom-turbopack.

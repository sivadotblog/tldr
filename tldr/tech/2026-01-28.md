[### Try text scaling support in Chrome Canary (7 minute read)](https://www.joshtumath.uk/posts/2026-01-27-try-text-scaling-support-in-chrome-canary/?utm_source=tldrnewsletter)

The new 'meta name=text-scale' HTML tag informs browsers that websites are designed to still work if users increase their OS text size. To support text scaling, developers should avoid overriding the initial font size, only use font-relative units for content, and test on devices to quickly figure out problem areas and think about how to adapt their layouts. The tag is currently supported in Chrome Canary with the 'Experimental Web Platform features' flag enabled. Support in other browsers will hopefully come later this year.

[### One Human + One Agent = One Browser From Scratch (2 minute read)](https://simonwillison.net/2026/Jan/27/one-human-one-agent-one-browser/#atom-everything?utm_source=tldrnewsletter)

one-agent-one-browser is a web browser built in three days with a single Codex CLI agent. It contains 20,000 lines of Rust with no Rust crate dependencies at all, with only OS system frameworks for image and text rendering. The browser successfully renders HTML and CSS and can even display SVGs and PNGs. The project proves that an agent driven by a talented engineer is enough to get a very solid basic renderer working.

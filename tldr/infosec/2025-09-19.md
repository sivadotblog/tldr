### Strategies & Tactics

[### Detect Secrets in GitLab CI Logs using ggshield and Bring Your Own Source (4 minute read)](https://blog.gitguardian.com/detect-secrets-in-gitlab-ci-logs/?utm_source=tldrinfosec)

GitGuardian demonstrates how to automatically scan GitLab CI/CD pipeline logs for secrets using ggshield and their Bring Your Own Source initiative, addressing a critical but often overlooked attack vector where sensitive data like environment variables, API responses, and temporary credentials are exposed in build logs. The solution involves creating a custom Docker image with enhanced ggshield capabilities, configuring secure token storage via HashiCorp Vault, and implementing an automated scanning script that processes all pipeline job logs and creates incidents in the GitGuardian dashboard. This approach provides comprehensive secrets detection across the entire DevOps lifecycle, catching runtime secrets that traditional source code scanning misses while maintaining zero friction for developers.

[### AWS AgentCore: The Overlooked Privilege Escalation Path in Bedrock's AI Tooling (6 minute read)](https://sonraisecurity.com/blog/aws-agentcore-privilege-escalation-bedrock-scp-fix/?utm_source=tldrinfosec)

AWS Bedrock AgentCore is a collection of tools that assist in deploying and scaling AI agents. AgentCore provides the ability to launch Code Interpreters, which run code in a sandboxed environment and can be customized to have access to AWS credentials to perform actions on AWS resources. Custom Code Interpreters cannot be protected with resource policies and only emit CloudTrail data events, presenting a tempting privilege escalation vector for attackers.

[### Lateral Movement - BitLocker (8 minute read)](https://ipurple.team/2025/08/04/lateral-movement-bitlocker/?utm_source=tldrinfosec)

BitLocker is a full disk encryption feature for Windows that protects data at rest by encrypting full volumes. This post outlines a privilege escalation or lateral movement technique that uses Windows Management Instrumentation (WMI) to hijack specific COM objects of BitLocker using remote manipulation of BitLocker registry keys to execute code under the user's context on a target host. The post also discusses detection opportunities for this attack and Sigma and Windows Defender for Endpoint queries to detect them.

WatchGuard has released security updates to address a critical (CVSS 9.3) remote code execution vulnerability impacting its Firebox firewalls. [### Meet ShadowLeak: ‚ÄòImpossible to detect' data theft using AI (8 minute read)](https://www.csoonline.com/article/4059606/meet-shadowleak-impossible-to-detect-data-theft-using-ai.html?utm_source=tldrinfosec)

Researchers at Radware discovered a vulnerability called ShadowLeak that exploits OpenAI's ChatGPT through indirect prompt injection attacks. Hidden email instructions can trick the AI into stealing sensitive data like credit card numbers and names from users' inboxes. The attack bypasses ChatGPT's security protections by encoding stolen data in Base64 and exfiltrating it directly through OpenAI's servers via the agent's browsing tool, making it nearly impossible to detect since it doesn't go through the client interface. This attack represents a significant escalation from previous client-side prompt injection attacks, as the data theft occurs entirely on the backend without any visible signs to the user.

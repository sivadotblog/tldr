### Attacks & Vulnerabilities

[### ERMAC V3.0 Banking Trojan Source Code Leak Exposes Full Malware Infrastructure (2 minute read)](https://thehackernews.com/2025/08/ermac-v30-banking-trojan-source-code.html?utm_source=tldrinfosec)

Complete source code for Android banking Trojan ERMAC 3.0 leaked from an open directory, revealing a Malware as a Service (MaaS) infrastructure targeting over 700 banking, shopping, and cryptocurrency apps. The leak exposed critical security flaws, including hardcoded JWT secrets, static admin tokens, and default credentials in the malware's PHP/Laravel backend and React frontend. ERMAC 3.0 features enhanced form injection methods, AES-CBC encrypted communications, and a Kotlin-based Android backdoor that avoids infecting devices in countries in the Commonwealth of Independent States (CIS).

[### Plex Warns Users to Patch Security Vulnerability Immediately (2 minute read)](https://www.bleepingcomputer.com/news/security/plex-warns-users-to-patch-security-vulnerability-immediately/?utm_source=tldrinfosec)

Plex has reached out to some of its users and urged them to update their Plex Media Server installation due to a recently patched security vulnerability. Plex has not assigned a CVE to the vulnerability or released details yet, but has stated that it was discovered via the Plex bug bounty program.

[### UK telecom firm Colt suffers massive ransomware attack (3 minute read)](https://cybernews.com/cybercrime/uk-telecom-firm-colt-suffers-massive-ransomware-attack/?utm_source=tldrinfosec)

The Warlock ransomware gang targeted the UK telecommunications firm Colt, causing disruptions to its online and voice services. In response, Colt took its systems offline, resulting in service interruptions, and is currently in the process of restoring operations. The attackers have claimed to have stolen sensitive data, potentially exploiting a flaw in Microsoft SharePoint.

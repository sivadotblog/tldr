[### Exploiting zero days in abandoned hardware (10 minute read)](https://blog.trailofbits.com/2025/07/25/exploiting-zero-days-in-abandoned-hardware/?utm_source=tldrinfosec)

Trail of Bits researchers exploited two discontinued home network devices (Netgear WGR614v9 router and BitDefender Box V1) at DistrictCon's Junkyard competition, achieving remote root access through chained vulnerabilities, including UPnP authentication bypass, buffer overflows, and firmware downgrade attacks. The research highlights how end-of-life hardware becomes a persistent security risk when manufacturers stop releasing updates, leaving vulnerabilities permanently unpatched. Both devices were initially designed to protect home networks, but became easy targets due to years without security updates.

### Launches & Tools

[### EntraGoat (GitHub Repo)](https://github.com/semperis/entragoat?utm_source=tldrinfosec)

EntraGoat is a deliberately vulnerable Microsoft Entra ID infrastructure that simulates real-world identity security misconfigurations and attack vectors. It introduces intentional vulnerabilities in your environment to provide a realistic learning platform for security professionals. EntraGoat features multiple privilege escalation paths and focuses on black-box attack methodologies.

[### Fable Security (Product Launch)](https://fablesecurity.com/?utm_source=tldrinfosec)

Fable uses AI to monitor real-time employee behavior signals and provides interventions such as chats and video briefings tailored to individual risk moments. It aims to educate users and reduce human-related cybersecurity threats in enterprises.

[### DoubleTeam (GitHub Repo)](https://github.com/ricardojoserf/DoubleTeam?utm_source=tldrinfosec)

DoubleTeam is a Python listener that uses socat, tmux, and threading to launch a new tmux window on each connection, listen on multiple ports, and automatically resume listening upon spawning the new window.

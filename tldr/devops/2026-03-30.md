### Opinions & Tutorials

[### Build a News Roundup With Docker Agent (4 minute read)](https://www.docker.com/blog/building-a-news-roundup-with-docker-agent-docker-model-runner-and-skill/?utm_source=tldrdevops)

A Principal Solutions Architect at Docker built a local AI workflow that uses Docker Agent with Qwen3.5-4B (a 4-billion-parameter model) and the Brave Search API to automatically generate IT news roundups without consuming cloud AI credits. The system uses a custom "news-roundup" skill that fetches articles via Brave Search, analyzes them locally through Docker Model Runner, and produces structured Markdown reports—trading speed for cost savings and local processing compared to cloud-based solutions like Claude.

[### How we use Abstract Syntax Trees (ASTs) to turn Workflows code into visual diagrams (8 minute read)](https://blog.cloudflare.com/workflow-diagrams/?utm_source=tldrdevops)

Cloudflare launched visual diagrams for its Workflows durable execution engine, using Abstract Syntax Trees (ASTs) and the oxc-parser to automatically generate flowcharts from minified JavaScript code that show how steps connect, branch, and execute in parallel. The company built a Rust Worker that parses deployed workflow code at build time to statically derive execution graphs, tracking Promise and await relationships to visualize the dynamic execution model where unawaited steps run in parallel by default.
